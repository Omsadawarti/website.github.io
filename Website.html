Topic:- HTML


HTML stands for hyper text markup language – it is used to display web pages on the browser. In order to create web pages, one should learn HTML. As its name suggests, HTML is a markup language, not a programming language. This means that we do not write programs using HTML; instead, markup language is mainly used to apply layout and formatting conventions to a text document. In other words, markup language makes text more interactive and dynamic.

HTML Tags:-

One of the main parts of HTML is HTML tags – HTML tags are like keywords that define how a web browser will format and display content. There are more than 100 tags in HTML.
